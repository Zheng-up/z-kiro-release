## 下载方式

### 🖥️ Windows
- `Z-Kiro-*.exe` 便携版，双击即用。  
> **SmartScreen 提示**：若出现 “Windows 已保护你的电脑 / Microsoft Defender SmartScreen 阻止了无法识别的应用启动”，点击 **“更多信息” → “仍要运行”**。

### 🍎 macOS
- `Z-Kiro-*-universal.dmg` 通用安装包，支持 Intel 和 Apple Silicon。  
> **“应用已损坏 / 来自身份不明的开发者” 提示**：在终端执行  
>
> sudo xattr -cr /Applications/Z-Kiro.app
> sudo spctl --master-disable
> > 参考 [这篇说明](https://www.cnblogs.com/cnndevelop/p/18702837) 解除限制后，再打开应用。

### 🐧 Linux
- `Z-Kiro-*.AppImage` 通用版（大多数发行版双击即可运行）  
- `z-kiro-desktop_*.deb` 适用于 Debian / Ubuntu  
- `z-kiro-desktop-*.rpm` 适用于 CentOS / RHEL / Fedora
