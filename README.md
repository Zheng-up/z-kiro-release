## 下载方式

### 🖥️ Windows
- `Z-Kiro-*.exe` 便携版，解压即用。

### 🍎 macOS
- `Z-Kiro-*-universal.dmg` 通用安装包，支持 Intel 和 Apple Silicon。

### 🐧 Linux
- `Z-Kiro-*.AppImage` 通用版（大多数发行版双击即可运行）
- `z-kiro-desktop_*.deb` Debian / Ubuntu
- `z-kiro-desktop-*.rpm` CentOS / RHEL / Fedora
